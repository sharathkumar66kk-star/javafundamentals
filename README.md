# javafundamentals
Structured java learning from basics 

This repository contains my step-by-step learning of Java fundamentals.
Each topic includes source code, explanation, and output.

## What this repository covers
- Java basics (variables, data types, operators)
- Control flow (if-else, switch, loops)
- Methods (basics and parameters)
- Introduction to OOP (class and object)


## Learning rule
Every topic follows this structure:
- Java source code
- Notes / explanation
- Output or test cases
