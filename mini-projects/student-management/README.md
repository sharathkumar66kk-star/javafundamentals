# Student Management System (Mini Project)

A lightweight, console-based Java application designed to manage student records efficiently using Object-Oriented Programming (OOP) principles.

## 📌 Project Overview
This project serves as a digital directory for educational institutions. It replaces manual paperwork by allowing users to programmatically add and display student information (ID, Name, and Age) in a structured format.

## 🚀 Key Features
- **Dynamic Storage:** Uses `ArrayList` to handle a growing number of student records without fixed limits.
- **Data Encapsulation:** Implements private fields and public getters to ensure data security.
- **Modular Architecture:** Separates data models (`Student`), business logic (`StudentService`), and execution (`Main`) for better maintainability.

## 🛠️ Technologies Used
- **Language:** Java (JDK 8 or higher)
- **Concepts:** OOP, Collections Framework (ArrayList), Packaging.

## 📂 Project Structure
- `Student.java`: The blueprint for student objects.
- `StudentService.java`: Contains the logic for adding and listing students.
- `Main.java`: The entry point that runs the application.
